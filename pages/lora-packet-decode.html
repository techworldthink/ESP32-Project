Lora Packet Decode

--Reference --

Install


        $ npm install lora-packet

Usage


        $ lora-packet-decode --hex 40F17DBE4900020001954378762B11FF0D

        $ lora-packet-decode --base64 QPF9vkkAAgABlUN4disR/w0=

Input from file


        $ lora-packet-decode --base64 "$( < binary.text)"
        
        $ lora-packet-decode --base64 "$(python3 read_data.py)"
