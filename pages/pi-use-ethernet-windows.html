Connect your Raspberry Pi to your computer via Ethernet (or How to Use Internet Connection Sharing on Windows 10)

-- Resource--

Requirements

  • RJ45 Cable
  • Wifi Adapter
  • Pi
  • Windows 10 machine
  • Hotspot (i use mobile Hotspot)

Steps

  1. Connect Wi-Fi adapter to windows PC. Then connect to internet.
  2. Connect your Raspberry Pi using the PC Ethernet Port using a straight Ethernet Cable.
  3. Open Network and Sharing Center ...
  4. Open Wi-Fi Properties ...
  5. In the “Wi-Fi Properties” window, click the “Sharing” tab. Check the box for “Allow other network users to connect through this computer’s Internet connection”. In the drop down menu below it, select the Ethernet connection. You’ll also want to check the box for “Allow others network users to control or disable the shared Internet connection”. Click “OK” .
  6. Open your Ethernet connection properties.
  7. Double click on "Internet Protocol Version 4 (TCP/IPv4)"
  8. Set -> IP : 192.168.137.1 (may be VirtualBox use this IP then choose 192.168.137.2 )
  9. Set -> Subnet Mask : 255.255.255.0.
  10. Its Ok Now!
