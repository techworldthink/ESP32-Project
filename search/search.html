How to install ESP32 Board in Arduino IDE ? 1. Open your Arduino IDE, go to File>Preferences

2. Put this Link into the “Additional Board Manager URLs” field as shown in the figure below. Then, click the “OK” button

Note: if you already have another boards (i.e ESP8266 boards URL), you can separate the URLs with a comma.

3. Then go to the Boards Manager. Go to Tools > Board > Boards Manager

4. Go to the search bar and type ESP32 then install ESP32 by Espressif Systems

5. Wait a few seconds then you've installed it
